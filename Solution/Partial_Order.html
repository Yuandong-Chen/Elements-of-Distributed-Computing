1. Show that if P and Q are posets defined on set X, then so is P∩Q.

Ans: P∩Q⊆P, so property on irreflexive and transitive remains on intersection set. 

2. Show that for all posets P on set X, there exists a total order Q on X such that P⊆Q.

Ans: Things become hard when P is uncountable infinite. For different elements x, y in P, if they are comparable, we skipped it, otherwise, we random choose x < y or y < x, and update P. It becomes total order after adding uncountable relationship to P. But how could we add uncountable relationship to P? It is lazy computing, for example, whenever we choose different elements x, y from P, if they are incomparable, we just randomly choose x < y or y < x which makes any x, y from P comparable, so P becomes total order in the sense that choosing behavior is countable. The key part is why x < y or y < x relationship has no contradiction to the current poset P? If poset P has two incomparable different elements x and y, and adding relationship x < y makes P not poset anymore, there must exists some path makes y loop back to x by < relationship, which makes x, y comparable. Otherwise, P keeps transitive property.

3. Show that if C1 and C2 are down-sets for any poset (E, <), then so is C1∩C2.

Ans: C1∩C2⊆C1

4. Consider the poset defined by the divides relations on the set of positive integers. Show that this poset is a lattice.

Ans: least common multiple is the lowest upper bound.

5. The transitive closure of a relaionship R on a finite set can also be defined as the smallest transitive relation on S that contains R. Show that the transitive closure is uniquely defined. We use "smaller" in the sense that R1 is smaller that R2 if |R1| < |R2|.

Ans: Since R is finite, we only need to use O(N^2) time complexity algorithm to check every node pair (x, y) in the Hasse diagram if there is a path from x to y. If path exists, add (x, y) as new relationship. It is obvious enough that I leave out the proof.
