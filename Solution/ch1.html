1.1 Give advantage and disadvantages of a parallel programming model over a distributed system (message based) model

Ans:

adv: more efficient on single machine, shared memory and global clock.

disadv: hard to scale up to massive machines

 

1.2 Compare the mechanisms for synchornization and communication in parallel programming languages and distributed programming languages.

Ans:

Sync in Distributed: Make Phase Lock, not efficient. Failed sometime, or may impossible without any upperbound.

Communication in Distributed: message passing

Sync in parallel: Global variable. (lock)

Communication in parallel: Shared memory.
